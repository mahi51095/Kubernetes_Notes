## ğ·¾ï¸ Example 1: Basic DaemonSet (Runs on Worker Nodes Only)

```yaml

kind: DaemonSet
metadata:
  name: nginx
spec:
  selector:
    matchLabels:
      app: nginx
  template:
    metadata:
      labels:
        app: nginx
    spec:
      containers:
      - name: nginx
        image: nginx
        ports:
        - containerPort: 80
```
